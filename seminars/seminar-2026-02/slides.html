‚úçÔ∏è Efficient Prompting

  • üéØ Be specific about the task: ‚ÄúWrite a Stata do-file to ‚Ä¶‚Äù / ‚ÄúRefactor this R function to ‚Ä¶‚Äù

  • üß© Provide context: goal, dataset description, key variables, constraints (WB rules, packages, runtime)

  • üìå Define the expected output: table/figure format, file to modify, function signature, acceptance criteria

  • üîÅ Ask AI to summarize + clarify first: ‚ÄúRestate the task in your own words and ask any clarifying questions before implementing.‚Äù

  • üîç Ask for assumptions + checks: ‚ÄúList assumptions; add validation/tests; include edge cases‚Äù

  • ü™ú Iterate in small steps: request a minimal working change, then refine (don‚Äôt ‚Äúrewrite everything‚Äù)

  • üß† Force transparency: ‚ÄúExplain why this approach; give 1‚Äì2 alternatives with trade-offs‚Äù

  • üßæ Make it reproducible: ‚ÄúUse set seed; pin versions; avoid randomness; write comments; ask LLM to write comments‚Äù

  • üõë Set boundaries: ‚ÄúDon‚Äôt use external data; don‚Äôt print secrets; don‚Äôt change unrelated files‚Äù

Tip: Treat the assistant as a collaborator.

‚ö†Ô∏è Limitations and Remedies

  • ‚ö†Ô∏é Wrong-but-plausible outputs / hallucinations: code runs but logic is wrong

    ‚úì Verify and validate: ask the model to explain and justify the solution

  • ‚ö†Ô∏é Context limits: not all files/data are in context; too large projects.

    ‚úì Be explicit: state assumptions, expected inputs/outputs, and references

  • ‚ö†Ô∏é Outdated knowledge: suggested APIs/packages/options may have changed

    ‚úì Teach the model: provide references/links; ask it to learn

  • ‚ö†Ô∏é Over-reliance: erodes fundamentals; mistakes slip through unchallenged

    ‚úì Keep learning: ask for step-by-step reasoning; request alternatives and trade-offs

  • ‚ö†Ô∏é Confidentiality / security / privacy

    ‚úì Constrain context: exclude sensitive data; use .copilot-ignore

  • ‚ö†Ô∏é Reproducibility: answers can vary across sessions/models/settings

    ‚úì Make it reproducible: save prompts, use Git; create AI agents
